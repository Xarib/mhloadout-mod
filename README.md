# mhloadout-mod

## Setup
```bash
git clone https://github.com/Xarib/mhloadout-mod.git
cd mhloadout-mod
```

## Build

### Visual Studio
1. Just open the root folder. Wait for VS to setup. This will take a while.
1. Select one setting
    - x64-Debug
    - x64-RelwithDebInfo
    - x64-MinSizeRel
1. GOTO Build > Rebuild All
